# Credits

## Built On

JunimoServer builds on these projects:

- [Junimohost](https://github.com/JunimoHost/junimohost-stardew-server) - Original dedicated server concept
- [AlwaysOn Mod](https://github.com/funny-snek/Always-On-Server-for-Multiplayer) - Keep-alive logic
- [NetworkOptimizer](https://github.com/Ilyaki/NetworkOptimizer) - Network performance improvements
- [SMAPIDedicatedServer](https://github.com/ObjectManagerManager/SMAPIDedicatedServerMod/tree/main/DedicatedServer) - Headless server functionality
- [CabinStacker](https://github.com/reyqn/CabinStacker) - Cabin management strategies

Thanks to **ConcernedApe** for Stardew Valley and **SMAPI maintainers** for making mods possible.
