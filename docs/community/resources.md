# Resources

Useful links for Stardew Valley modding, related projects, and tools.

## Stardew Valley Modding

- [SMAPI](https://smapi.io/) — The modding framework that powers JunimoServer
- [SMAPI Documentation](https://stardewvalleywiki.com/Modding:Index) — Official modding wiki
- [Nexus Mods](https://www.nexusmods.com/stardewvalley) — Largest mod repository
- [Stardew Valley Discord](https://discord.gg/stardewvalley) — Community and modding help

## Related Projects

- [puppy-stardew-server](https://github.com/truman-world/puppy-stardew-server) — Alternative dedicated server implementation
- [pterodactyl-server-stardew-valley](https://github.com/paytah232/pterodactyl-server-stardew-valley) — Pterodactyl panel implementation

## Tools

- [Farmtronics](https://github.com/JoeStrout/Farmtronics) — In-game programmable bots
- [Stardew Predictor](https://mouseypounds.github.io/stardew-predictor/) — Predict RNG events
