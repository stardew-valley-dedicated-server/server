# Player Troubleshooting

Common connection problems and solutions.

## Can't Connect

| Symptom | Likely Cause | Solution |
|---------|--------------|----------|
| "Connection failed" | Wrong invite code | Verify code with admin |
| Immediate disconnect | Mod mismatch | Check you have all required mods |
| Can't connect from same network | Router doesn't support hairpinning | Use mobile hotspot or VPN |

## Password/Login Issues

| Symptom | Solution |
|---------|----------|
| "Invalid password" | Check for typos (case-sensitive) |
| Stuck in lobby | Type `!login <password>` |
| Kicked after failed attempts | Wait, reconnect, use correct password |

## In-Game Issues

| Symptom | Likely Cause | Solution |
|---------|--------------|----------|
| Missing items/buildings | Missing content mod | Ask admin for mod list |
| Invisible players/NPCs | Missing content mod | Install required mods |
| Chat not working | Wrong key | Press `T` or `/`, commands start with `!` |
| High ping/lag | Network issues | Check connection, use `!info` to see ping |

## Getting Help

1. **Ask your server admin** — They can check logs
2. **Join Discord** — [discord.gg/w23GVXdSF7](https://discord.gg/w23GVXdSF7)

When asking for help, include: error message, your platform (Steam/GOG), and whether you have mods.
