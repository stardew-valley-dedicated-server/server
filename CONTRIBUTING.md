## Quick Start

1. Create a feature branch from `master`
2. Make changes using [Conventional Commits](https://www.conventionalcommits.org/)
3. Open a PR targeting `master`
4. Wait for CI validation and review

For detailed instructions, see the [full contribution guide](docs/4.community/3.contribution.md).
